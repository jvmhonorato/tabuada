## Tabuada




Application that shows multiplication tables from 1 to 100 


## Technologies 

Here are the technologies used in this project.

* NODE JS
* HTML
* CSS





## Services Used

* Github





## Getting started

* Dependency
  - Express JS
  - Axios
  - EJS
  - mocha
  - sinon
 
  
  
* To install the dependencies.
  - npm install
  
* To run the project.
  -npm  start
